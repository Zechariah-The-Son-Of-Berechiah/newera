## ğŸ“ Customization

### Colors
Edit CSS variables in `css/main.css`:
```css
:root {
    --primary: #0f0f0f;
    --secondary: #1a1a1a;
    --accent: #FFC107;
```

- Update text content in `index.html`
- Replace images in the `images/` folder
- Modify countdown date in `js/countdown.js`

- Style new components in `css/components.css`
